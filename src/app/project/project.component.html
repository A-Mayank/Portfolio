Healthcare Accessibility App

Find nearby medical facilities & chat with an AI for symptom analysis.


Project Overview

This project aims to improve access to healthcare by providing users with a seamless platform to locate nearby medical facilities and assess their symptoms through an AI-powered chatbot. The application integrates Google Maps API to pinpoint hospitals, clinics, and pharmacies based on the user's real-time location, ensuring quick and reliable access to healthcare services.

Features

  • Utilizes Google Maps API to fetch and display nearby hospitals, clinics, and pharmacies.
  • Provides details such as hospital name, address, contact information, and directions.
  • Features a search function to filter results based on user preferences.

AI-Powered Symptom Assessment Chatbot

  • Built using NLTK (Natural Language Toolkit)for Natural Language Processing (NLP).
  • Allows users to input their symptoms and receive preliminary health insights.
  • Offers guidance on whether to seek medical attention based on symptom severity.

Technologies Used

  • Backend: Flask(Pyhton)
  • Frontend: HTML, CSS, JavaScript.
  • AI/NLP: NLTK(Natural Language Toolkit)
  • API's Used: Google Maps API
